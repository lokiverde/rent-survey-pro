# Rent Survey Prompt Kit

## What This Is

A standalone prompt you can paste into ANY AI assistant (ChatGPT, Claude, Gemini, etc.) to get a professional rent survey.

No setup required. Just copy, paste, fill in the blanks.

## How to Use

1. Open your AI assistant of choice
2. Make sure it has web browsing enabled (for current data)
3. Copy the prompt from [rent-survey-prompt.md](./rent-survey-prompt.md)
4. Fill in your property details
5. Get your rent survey

## When to Use This vs. Custom GPT/Gem

**Use the prompt kit when:**
- You only need one rent survey
- You don't want to set anything up
- You want to try it before building a custom version

**Use a Custom GPT or Gem when:**
- You do rent surveys regularly
- You want a dedicated tool
- You want to share it with your team
